Wirewerks
========

Setup
-------

**Web client**
```
cd wirewerks\client\ui
npm install
cd src
npm install
```

Start serving the ui

```
cd wirewerks\client\ui
npm start
```

**Lib sass (for css)**

Sass files are build to .css by Intellij.

```
cd wirewerks
cd server
cd ww-libsass
npm install
cd node_sass
npm install
```
